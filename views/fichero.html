En cascada


Identifique la primera palabra y la ultima de cada bloque con las pistas que se dan. A continuación,
trate de descubrir las palabras intermedias. Para lograrlo, cambie una letra de la primera palabra
para obtener la próxima y, después, altere el orden de una o varias letras para encontrar la siguiente.
Siga así. sucesivamente, hasta que logre completar ambos casilleros. Todas las palabras intermedias
deben tener signidicado.
